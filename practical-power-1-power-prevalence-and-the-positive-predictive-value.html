Practical power 1: Power, prevalence, and the positive predictive value

Date 01 October 2019 Modified 01 October 2019 By Johan Carlin Tags scientific computing / python / power / statistics

A major cause of the replication crisis that currently plagues the life sciences is low statistical power. But power is a nebulous concept, and power calculations are rarely used to plan experiments (and still more rarely used sensibly). In this notebook I introduce basic power calculation terminology, and cover how the positive and negative predictive value quantifies how statistical power and significance thresholding influences reproducibility. Stay tuned for the next notebook, where we discuss effect sizes and sample size calculations.

I work in human neuroimaging and psychology. Researchers in these fields are increasingly concerned about reproducibility. Informally, that means we worry that a lot of published claims aren't true. We can formalise this intuition in terms of the positive predictive value (PPV), that is, the proportion of reported statistically significant effects that represent a true effect in the population that the sample came from. Ioannidis (2005) famously argued that if we estimate the PPV of an entire literature, there are possible scenarios where PPV is less than 0.5. That is, most published significant effects in that literature are actually false positives. It's difficult to imagine that cumulative science can happen under such conditions (especially if non-significant results are left in the file drawer rather than reported). What's going wrong?

Poor reproducibility has two basic causes. First, questionable research practices such as the file drawer effect and the garden of forking paths [pdf] (sometimes more pejoratively described as p-hacking), which bias reported effect sizes and inferential statistics. I think most researchers are aware of these issues by now, although the cures (pre-registration, validation samples) have yet to be widely adopted. The second cause of poor reproducibility is less widely understood: low statistical power, and overly-liberal significance thresholding. This notebook is an attempt to make this more concrete. I will start with a refresher on definitions, but feel free to skip ahead to the code, where we explore expected reproducibility in a few scenarios.

A quick review of alpha, power, and predictive values¶

Terminology¶

Here is a little reminder of a classic null hypothesis testing contingency table, with common labels for each cell:

statistical test null true null false
significant $\alpha$ false positive rate, false alarm $1-\beta$ power, sensitivity, recall, hit
not significant $1-\alpha$ specificity, precision, correct rejection $\beta$ false negative rate, miss
  • $p$ is the probability of observing a test statistic of this size or greater when the null hypothesis is true
  • $\alpha$ is the significance threshold we adopt for $p$ (conventionally 0.05). We reject the null hypothesis when $p < \alpha$. We generally refer to this as the false positive rate, because we expect to report a false positive on $\alpha$ proportion of the tests we calculate when the null hypothesis is true
  • $\beta$ is the false negative rate. It represents the proportion of statistical tests that we expect to be non-significant for a given effect size and $\alpha$ when the null hypothesis is false
  • $1-\beta$ is the power of a statistical test (often fixed at 0.8 for sample size calculations). It indicates how often an experiment is expected to produce a statistically significant effect for a given effect size when the null hypothesis is false

The false positive rate $\alpha$ and false negative rate $\beta$ tells us how often we expect to get the wrong answer given that the null or the alternative hypothesis is true, respectively. What's missing from this analysis is the base rate, that is, what proportion of the hypotheses we test are expected to be true? As we will see next, this $prevalence$ parameter is crucial for fully characterizing the expected properties of our hypothesis testing approach.

Positive predictive value (PPV) and prevalence¶

To calculate the PPV, that is, the proportion of all reported significant tests that are true positives, we need one more thing: $prevalence$ is the proportion of tested hypotheses that are true. If $prevalence=1$, all hypotheses are true, and any non-significant test is a false negative. If $prevalence=0$, all hypotheses are false, and all significant tests are false positives. Once we have fixed $prevalence$, PPV is simply the true positives scaled by the sum of the true and false positives:

$$\text{PPV} = \frac{(1-\beta) \times \text{prevalence}}{(1-\beta) \times \text{prevalence}+\alpha \times (1-\text{prevalence})}$$

On the numerator, recall that $(1-\beta)$ is power, ie the proportion of true hypotheses where we expect to obtain a statistically significant test. We need to scale this by prevalence to consider how common true hypotheses are. The denominator is the sum of this and the false positive rate $\alpha$ scaled by how often we expect to encounter false hypotheses (the remaining of the cases, so $(1-prevalence)$).

At this point, you might be wondering what a reasonable value for $prevalence$ would be. We don't know! It might be 0 in parapsychology, and close to 1 for a lot of other areas of psychology (for instance, it's sometimes hard to imagine that stated hypotheses in applied or health psychology could be false). We'll return to this point in the code below.

Negative predictive value (NPV)¶

Negative predictive value is basically the same calculation as PPV, but applied to the lower row of the null hypothesis testing contingency table above. That is, when we fail to reject the null hypothesis, how often does this reflect a true null effect?

$$\text{NPV} = \frac{(1-\alpha) \times (1-\text{prevalence})}{(1-\alpha) \times (1-\text{prevalence}) + \beta \times \text{prevalence}}$$

On the numerator we need the specificity of the test $(1-\alpha)$ scaled by the proportion of true nulls $(1-prevalence)$. The denominator is the sum of this and the false negative rate scaled by the proportion of true hypotheses ($\beta \times \text{prevalence}$).

Scientists are often trained to avoid claiming that an effect is absent when a test is non-significant, which perhaps explains why NPV is discussed less often than PPV. But NPV provides a useful complimentary perspective on power and prevalence. As we'll see next, a high PPV by itself does not guarantee that a field is in good shape.

I don't know this is starting to sound a bit Bayesian¶

Why yes! In Bayesian terms, PPV is the posterior probability of a true hypothesis given a significant test, and 1-NPV is the posterior probability of a true hypothesis given a non-significant effect (Johnson et al, 2001 - thanks to Will Penny for bringing this connection to my attention). It turns out that we can't avoid prior belief (ie, the prevalence parameter) if we want to estimate PPV and NPV. But proper Bayesians would naturally object to the premise of discretizing evidence into arbitrary 'significant' and 'non-significant' categories, when we could be characterizing the entire posterior distribution instead... We may return to Bayesian alternatives to power analysis in a future post.

Exploring positive and negative predictive values in different scenarios¶

The conclusion that most published findings are false in Ioannidis (2005) holds specifically when $prevalence<0.5$, power is low, and statistical significance is assessed at the conventional $p<0.05$ level. Whether that's a reasonable description of the state of your research area is for you to judge. To make this more concrete, let's explore the PPV and NPV consequences of different power and prevalence combinations.

In [2]:
# in python, PPV and NPV can be calculated as
# notice that we use power (1-beta) as input instead of beta here
ppv = lambda power, alpha, prevalence: (power * prevalence) / \
    (power * prevalence + alpha * (1 - prevalence))
npv = lambda power, alpha, prevalence: ((1-alpha) * (1-prevalence)) / \
    ((1-alpha) * (1 - prevalence) + (1-power) * prevalence)

# we can also visualise the scenario with a pie chart like so:
def power_pie(power, alpha, prevalence):
    # we can either 'hit' or 'miss' true positives
    hits = power * prevalence
    misses = (1-power) * prevalence
    # and for true positives, we can have false positives and correct rejections
    false_positives = alpha * (1-prevalence)
    correct_rejections = (1-alpha) * (1-prevalence)
    ax = plt.gca()
    ax.pie((hits, misses, false_positives, correct_rejections),
           labels=(f"H1, p<{alpha}", f"H1, p>{alpha}", f"H0, p<{alpha}", f"H0, p>{alpha}"),
           colors=("darkslateblue", "slateblue", "darkgoldenrod", "goldenrod"))
    return

Scenario 1¶

Let's say we are doing discovery science (only 25% of hypotheses are true) and let's further assume we are good scientists, who power studies to the conventional 80% level:

In [3]:
good = ppv(power=0.8, alpha=0.05, prevalence=0.25)
print(f"well-powered discovery scientists are right {good*100:.3}% "
      "of the times they declare significance")
good_neg = npv(power=0.8, alpha=0.05, prevalence=0.25)
print(f"...and are right {good_neg*100:.3}% "
      "of the times they fail to declare significance")
fig, ax = plt.subplots(1, 1, figsize=[2, 2])
power_pie(power=0.8, alpha=0.05, prevalence=0.25)

well-powered discovery scientists are right 84.2% of the times they declare significance
...and are right 93.4% of the times they fail to declare significance

Not too bad, right?

Scenario 2¶

Now suppose we work in the same field, but we are 'bad' scientists who haven't powered our studies appropriately:

bad = ppv(power=0.2, alpha=0.05, prevalence=0.25)
print(f"underpowered discovery scientists are right {bad*100:.3}% "
      "of the times they declare significance")
bad_neg = npv(power=0.2, alpha=0.05, prevalence=0.25)
print(f"...and are right {bad_neg*100:.3}% "
      "of the times they fail to declare significance")
fig, ax = plt.subplots(1, 1, figsize=[2, 2])
power_pie(power=0.2, alpha=0.05, prevalence=0.25)

underpowered discovery scientists are right 57.1% of the times they declare significance
...and are right 78.1% of the times they fail to declare significance

We are now perilously close to that <0.5 PPV that led Ioannidis (2005) to claim that most published findings are false. Notice that NPV remains decent because the prevalence of true hypotheses is low (you can't miss something that isn't there to detect).

Scenario 3¶

Let's next imagine we are good scientists with adequate power, but we work in a field where almost every hypothesis is false (let's call it social priming). For this, we will set the prevalence of true hypotheses much lower:

In [5]:
misguided = ppv(power=0.8, alpha=0.05, prevalence=0.05)
print(f"well-powered but misguided scientists are right {misguided*100:.3}% "
      "of the times they declare significance")
misguided_neg = npv(power=0.8, alpha=0.05, prevalence=0.05)
print(f"...and are right {misguided_neg*100:.3}% "
      "of the times they fail to declare significance")
fig, ax = plt.subplots(1, 1, figsize=[2, 2])
power_pie(power=0.8, alpha=0.05, prevalence=0.05)

well-powered but misguided scientists are right 45.7% of the times they declare significance
...and are right 98.9% of the times they fail to declare significance

Even with adequate power, chasing sufficiently rare effects results in low PPV. How can you preserve a high PPV in such a scenario? As we will see below, the answer turns out to be adopting a stricter criterion for significance ($\alpha$).

Scenario 4¶

Finally, let's suppose we are underpowered, but we work in a field where most of the hypotheses that are being tested are likely to be true anyway (let's call it applied psychology). For this, we set power low and prevalence high:

In [6]:
obvious = ppv(power=0.2, alpha=0.05, prevalence=0.75)
print(f"underpowered scientists of the obvious are right {obvious*100:.3}% "
      "of the times they declare significance")
obvious_neg = npv(power=0.2, alpha=0.05, prevalence=0.75)
print(f"...and are right {obvious_neg*100:.3}% "
      "of the times they fail to declare significance")
fig, ax = plt.subplots(1, 1, figsize=[2, 2])
power_pie(power=0.2, alpha=0.05, prevalence=0.75)

underpowered scientists of the obvious are right 92.3% of the times they declare significance
...and are right 28.4% of the times they fail to declare significance

Notice that low power does not necessarily produce low PPV, if you're mostly making 'safe' predictions. But the price you pay for low power in this scenario is catastrophic NPV - there are lot of other hypotheses that are also true, but you can't detect them.

Taken together, these calculations illustrate that:

  • Low statistical power affects both PPV and NPV. It's a common misconception that we don't have to worry about power as long as we have achieved statistical significance (for an example of this type of reasoning, see Friston, 2012). But as we have seen, low power means lower NPV and PPV. If we suspect that power is low, we should be less confident in both significant and non-significant results.
  • If you work in a field where the prevalence of true effects is low, high statistical power won't save you from low PPV. Even if you are detecting true effects resonably well when they are there, PPV ends up dominated by the false positives. We should be skeptical of significant effects from 'high-risk' experiments that test wildly implausible hypotheses.
  • If you work in a field where null hypothesis tests are reported as 'sanity checks' for effects that everyone already basically believes in (rather than to claim a genuine new discovery), your PPV can remain quite high even if power is low. But the price you pay is low NPV. A researcher in such a field might have the frustrating experience that experiments that really 'should' work often won't.

The relationship between prevalence, significance thresholds, and power¶

One general theme that emerges from these calculations is that if we adopt current standards (80% power and a 5% false positive rate), we will only observe high PPV and NPV in a true effect prevalence 'sweet spot'.

However, other investigators have proposed adopting a stricter $p<0.005$ criterion for significance (Benjamin et al, 2017). How would that change the position of the 'sweet spot'? And what if statistical power in your field is actually more like 20%, as has been claimed for neuroscience research (Button et al, 2013)? Let's plot out PPV and NPV against prevalence for a few scenarios.

In [7]:
prevalence_rates = np.linspace(0, 1, 100)
fig, ax = plt.subplots(2, 2, sharex=True, sharey=True, figsize=(4,3))
alpha_rates = iter((.05, .05, .005, .005))
power_rates = iter((.2, .8, .2, .8))

for this_row in ax:
    for this_ax in this_row:
        this_alpha = next(alpha_rates)
        this_power = next(power_rates)
        this_ppv = ppv(power=this_power, alpha=this_alpha, prevalence=prevalence_rates)
        this_npv = npv(power=this_power, alpha=this_alpha, prevalence=prevalence_rates)
        this_ph = this_ax.plot(prevalence_rates, np.hstack((this_ppv[:,None], this_npv[:,None])))
        sweet_spot = prevalence_rates[np.argmin(np.abs(this_ppv - this_npv))]
        this_sp = this_ax.plot([sweet_spot, sweet_spot], [0., 1.], ":k")
        this_st = this_ax.text(sweet_spot, 1.1, f"{sweet_spot:.2f}", horizontalalignment="center")
        if np.all(this_row == ax[-1]):
            this_ax.set_xlabel(f"prevalence\n(power={this_power})")
            if this_ax == this_row[-1]:
                this_ax.legend(this_ph, ("PPV", "NPV"))
    this_row[0].set_ylabel(f"rate\n($\\alpha$={this_alpha})")

There's a lot going on in these plots. Let's unpack:

  • That prevalence 'sweet spot' we saw a hint of in the examples above is the rate at which the PPV (blue) and NPV (yellow) curves intersect (dotted line, prevalence value on upper margin). Assuming that we want to maximize both PPV and NPV, this is the prevalence rate that is optimal for a particular power and $\alpha$. Equal weights seems reasonable for discovery science, but if our priorities are different we could of course calculate a sweet spot for some other weighting.
  • Increasing power leads to higher PPV and NPV, but does not shift the position of the sweet spot (in columns). Even with adequate power, PPV falls off precipitously as prevalence drops below 0.15. As we also saw in the 'misguided' example scenario above, adequate statistical power by itself will not save us if effect prevalence is low.
  • Setting a more stringent $\alpha$ (0.05 to 0.005, in rows) shifts the prevalence sweet spot to lower rates (from 0.35 to 0.15). This seems desirable if we suspect that effect prevalence is indeed low (Benjamin et al, 2017 discuss evidence for a rate of 0.1 in psychology). Note that this change is almost entirely drively by PPV improvements. NPV is scarcely affected by $\alpha$ in this range.
  • Keep in mind that adjusting $\alpha$ while maintaining power typically means increasing sample sizes! It's probably unrealistic to think that Benjamin et al's (2017) proposed new significance level would result in larger samples being collected, since most researchers plan sample size by heuristics rather than by formal power analysis (in my field, N=20 is an unshakeable dogma, regardless of the question at hand). To roughly approximate the effect of changing $\alpha$ by itself, we can compare the top right ($\alpha=0.05, (1-\beta)=0.8$) and bottom left ($\alpha=0.005, (1-\beta)=0.2$) panels. This is a conservative approximation because changing $\alpha$ from .05 to .005 won't generally reduce power by this much. Even so, PPV improves with stricter $\alpha$. The price for setting a more stringent $\alpha$ without adjusting sample sizes will be paid in lower NPV.

Conclusions¶

I hope these conceptual calculations help clarify how to think about how power and $\alpha$ influences reproducibility. I'm going to close on a few take-home messages.

  • Low power means low trust in reported non-significant and significant tests.
  • It's dangerous to focus narrowly on PPV. Cumulative science with null hypothesis testing probably requires high NPV as well.
  • We can't reason about the reproducibility of a field (for instance, by calculating PPV and NPV) without setting the prevalence rate for true hypotheses. This need to express what amounts to prior beliefs can be uncomfortable for frequentists, but I hope you'll agree it's worthwhile.
  • High PPV and NPV is only expected within a restricted range of prevalence rates. If we suspect prevalence rates are lower than this, the solution is generally to adopt a stricter criterion for statistical significance ($\alpha$). If NPV is important, we will also have to maintain power at the same level despite the stricter $\alpha$, typically by increasing sample sizes (you could equally reduce measurement noise, but this attractive alternative is rarely available to us - in a random-effects group analysis of human data, between subject variance is often high no matter the refinement of your measurement instrument).

In the next notebook, we will turn to the more practical issue of sample size calculations. We will see that this requires an estimate of standardized effect size (e.g., cohen's $d$), and that obtaining an unbiased, low-variance estimate of this from previous studies is hard. Stay tuned for that.
